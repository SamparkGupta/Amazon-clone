# Amazon Clone

A front-end clone of the Amazon homepage built using HTML5 and CSS3. This project was created to practice modern web development concepts such as layout design, Flexbox, styling, positioning, and recreating real-world user interfaces.

## Technologies Used

## Features

* Amazon-inspired navigation bar
* Search bar interface
* Hero banner section
* Product showcase cards
* Multiple shopping categories
* Interactive hover effects
* Multi-column footer
* Clean desktop layout
* Built entirely without JavaScript
